🚀 EcomZone CMS

Installation & Setup Walkthrough

🗄️ Phase 1: Database Setup

1
Open MySQL Command Line
Open Windows Command Prompt or PowerShell:
mysql -u root -p
Leave password blank (press Enter) if no root password set.
2
Create Database
In MySQL prompt, run:
CREATE DATABASE cms_ecomzone; USE cms_ecomzone;
3
Import Schema
In MySQL prompt, run:
SOURCE c:\xampp\htdocs\EcomZone-CMS\database\cms_ecomzone.sql;
Wait for "Query OK" messages (18 tables created).
✅ Success Indicator: No errors. See message "Database created successfully with 18 tables and default admin user."

⚙️ Phase 2: Configuration

1
Verify Database Connection
Edit config/database.php. Ensure:
define('DB_HOST', 'localhost'); define('DB_USER', 'root'); define('DB_PASS', ''); // Leave blank if no password define('DB_NAME', 'cms_ecomzone');
2
Create Uploads Folder
Create folder: uploads/ with subfolders:
uploads/ ├── avatars/ ├── documents/ ├── invoices/ └── receipts/
Set permissions: Right-click → Properties → Security → Full Control for IUSR user.
ℹ️ Note: All configuration constants are in config/config.php. Customize timezone, currency, colors there.

🌐 Phase 3: Start & Access

1
Start XAMPP Services
In XAMPP Control Panel:
  • Click Start for Apache
  • Click Start for MySQL
2
Open Browser
Visit:
http://localhost/EcomZone-CMS/
You should see the purple login screen.
3
Login with Default Credentials
Field Value
Email admin@cms-ecomzone.com
Password Admin@123
You should land on the Dashboard.
✅ Congratulations! EcomZone CMS is installed and ready to use.

🆘 Troubleshooting

❌ "Can't connect to MySQL server"
→ Start MySQL from XAMPP Control Panel → Try again
❌ "No such table" during login
→ Database not imported properly. Repeat Phase 1, Step 3.
❌ "Permission denied" on file upload
→ Right-click uploads/ folder → properties → Security → Make writable
❌ Charts not showing on dashboard
→ Open browser console (F12) and check for JS errors → Reload page
